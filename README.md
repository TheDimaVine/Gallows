# Gallows (Виселица)
Консольная игра «Виселица», реализованная на языке Java.

## Описание
Игроку предлагается угадать слово, вводя буквы.
За каждую неверную попытку отрисовывается часть виселицы.
Количество попыток ограничено.

## Функциональность
- Загрузка слов из файла
- Проверка корректности ввода
- Учёт повторных букв
- Отрисовка виселицы
- Проверка победы и поражения

## Используемые технологии
- Java
- Collections Framework
- Работа с файлами (NIO)

## Запуск
1. Склонировать репозиторий
2. Запустить класс `Main`
